# ayed-step-intensive-2026

موقع **دورة STEP المكثفة 2026** (نسخة ويب ثابتة Static) — جاهز للرفع على GitHub وتشغيل GitHub Pages.

## محتويات المستودع
- `index.html` الصفحة الرئيسية
- مجلدات الموقع (CSS/JS/صور/صفحات) حسب البنية الحالية
- `.github/workflows/pages.yml` نشر تلقائي على GitHub Pages عبر GitHub Actions
- `.nojekyll` لتعطيل معالجة Jekyll

## تشغيله محليًا (على جهازك)
- افتح ملف `index.html` مباشرة في المتصفح.
- أو شغّل سيرفر محلي (اختياري) عشان تتأكد من الروابط:
  - VS Code: Extension **Live Server**

## رفعه على GitHub (خطوة بخطوة)
1) ادخل GitHub وسوِّ **New repository** باسم:
   - `ayed-step-intensive-2026`
2) ارفع كل ملفات هذا المستودع (Upload files) أو ارفعها عبر Git.
3) بعد أول Push لفرع **main**:
   - روح: **Settings → Pages**
   - تحت **Build and deployment** اختر:
     - **Source: GitHub Actions**
4) انتظر دقيقة… وبيطلع لك رابط الموقع من نفس صفحة Pages.

## ملاحظة مهمة
- هذا الموقع ثابت (Static). ما يرفع الملفات لسيرفر بشكل مباشر.
- نموذج التسجيل الحالي يجهّز رسالة منظمة للتواصل مع الحساب الرسمي حسب الإعدادات داخل الكود.

## تعديل بيانات سريعة
إذا احتجت تغيّر:
- تاريخ إغلاق التسجيل
- السعر / السعر الرسمي
- نصوص المحتوى

بتلقى أغلبها واضحة داخل ملفات JS/الصفحات.
